# 🏠 House Price Prediction

## 📌 Overview
This project predicts house prices using Machine Learning based on features like area, number of rooms, and location.

---

## 📊 Dataset
The dataset (house prices ) includes:
- Area
- Bedrooms
- Bathrooms
- Floors
- YearBuilt
- Location
- Condition
- Garage
- Price (target)

---

## ⚙️ Steps
- Load the dataset
- Explore data (head, info, describe)
- Clean data (check missing values)
- Encode categorical features
- Split data into train/test
- Train model (Random Forest Regressor)
- Evaluate using MAE, MSE, RMSE
